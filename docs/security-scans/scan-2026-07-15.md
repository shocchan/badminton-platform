## 6. 残存リスクの評価と受容判断

- **xlsx (SheetJS) の既知脆弱性（Prototype Pollution / ReDoS、修正版なし）**:
  当サイトでの用途は管理画面からの参加者リストの**書き出しのみ**（`AdminPage.tsx` の `writeFile`）。
  脆弱性が成立するのは「悪意ある .xlsx ファイルを読み込む」場合であり、読み込み機能は存在しないため**実質的に悪用不可。リスク受容とする**。
  読み込み機能を将来追加する場合は代替ライブラリへの移行を必須とする。
- vite / ws / undici の既知脆弱性は 2026-07-15 の `npm audit fix` で解消済み（いずれも開発ツール系）。
