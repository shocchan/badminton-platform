## 2. FAQ本文（日本語）— サイトFAQページ用

**Q. 初めてでも、一人でも参加できますか？**
A. はい。参加者の多くが一人でのお申し込みです。当日は運営がグループ分け・対戦の組み分けをサポートするので、知り合いがいなくても問題なく楽しめます。

**Q. 初心者でも大丈夫ですか？レベルはどのくらいですか？**
A. 初中級を中心に、幅広いレベルの方が参加しています。回によってシングルス・ミックスダブルスなど形式が変わるので、ご自身に合った回を選べます。ブランクからの復帰にもおすすめです。

**Q. 持ち物は？ラケットのレンタルはありますか？**
A. 動きやすい服装・体育館シューズ（室内用）・ラケット・飲み物をお持ちください。ラケットレンタルは【要記入：あり／なし】です。

**Q. 参加費と支払い方法を教えてください。**
A. 参加費は各大会ページに記載しています。PayPayでの事前決済に対応しており、当日精算が不要でスムーズに入場できます。

**Q. 会場はどこですか？アクセス・駐車場は？**
A. 川口・蕨エリアの体育館を中心に開催しています。最寄り駅・駐車場の有無は各大会ページに記載しています（駐車場：会場により◯台／近隣コインパーキング利用）。

**Q. 女性一人でも参加しやすいですか？**
A. はい。女性のお一人参加も多く、運営がフォローするので安心してご参加いただけます。

**Q. 中国語での対応はできますか？**
A. 中国語での問い合わせ・当日サポートに対応しています。WeChat・小红书でも情報を発信しており、中国語話者の参加者も多数います。

**Q. 当日キャンセルや欠席はできますか？**
A. キャンセルは各大会の締切までにお願いします。締切後のキャンセルは【要記入：キャンセルポリシー】となります。

**Q. どんな雰囲気ですか？何人くらい集まりますか？**
A. 交流を大切にした和やかな雰囲気です。毎回◯名前後が参加しています。試合だけでなく、バドミントン仲間づくりの場としても好評です。

**Q. 見学だけでも参加できますか？**
A. 【要記入：可／不可】。ご希望の場合は事前にお問い合わせください。

## 3. FAQ（中文）— 中華圏メンバー向けミラー（差別化ポイント）

**Q. 第一次参加、一个人也可以吗？**
A. 可以。很多参加者都是一个人报名的。当天由主办方安排分组和对战，即使没有认识的人也能安心参加。

**Q. 初学者可以参加吗？水平大概如何？**
A. 以初中级为主，各种水平的人都有参加。不同场次形式不同（单打・混双等），可以选择适合自己的场次。中断后想重新开始打球的人也很推荐。

**Q. 需要带什么？可以租球拍吗？**
A. 请带方便运动的服装、室内球鞋、球拍和饮用水。球拍租借【要記入：有／无】。

**Q. 参加费和付款方式？**
A. 参加费在各比赛页面标注。支持PayPay提前付款，当天无需现场结算，入场更顺畅。

**Q. 提供中文服务吗？**
A. 提供中文咨询和当天中文支持。我们也在微信・小红书上发布信息，有很多中文参加者。

**Q. 场地在哪里？有停车场吗？**
A. 主要在川口・蕨地区的体育馆举办。最近车站和停车场信息在各比赛页面标注。

  "endDate": "2026-08-【要記入】T【要記入】+09:00",
  "eventStatus": "https://schema.org/EventScheduled",

  "location": {
    "@type": "Place",
    "name": "【要記入：会場名】",
    "address": {
      "@type": "PostalAddress",
      "addressLocality": "川口市",
      "addressRegion": "埼玉県",
      "addressCountry": "JP"
    }
  },
  "organizer": {
    "@type": "Organization",
    "name": "川口・蕨バド交流杯",
    "url": "https://kawabado.com"
  },
  "offers": {
    "@type": "Offer",
    "price": "【要記入】",
    "priceCurrency": "JPY",
    "availability": "https://schema.org/InStock",
    "url": "https://kawabado.com/【大会ページURL】"
  },
  "description": "川口・蕨エリアで開催する社会人バドミントン大会。初心者・お一人参加も歓迎。PayPay事前決済対応。"

## 5. 「AIに読める形」チェックリスト

- [ ] 参加費が**テキスト**で書かれている（画像内文字になっていないか）
- [ ] 会場名・アクセス・駐車場がテキスト
- [ ] レベル感・形式（シングルス/ダブルス）がテキスト
- [ ] 見出し（h1/h2/h3）が「大会名」「参加費」「会場」等で構造化
- [ ] FAQが文章で書かれている（バナー画像だけになっていない）
- [ ] 「川口」「蕨」「バドミントン」「社会人」が本文に自然に入っている
- [ ] スマホで読みやすい／ページが重すぎない

## 実装メモ（Claude Code 用）

1. `/faq` ページ（or 各ページ下部FAQセクション）を作成し、上のQ&A本文を実装
2. 同ページに FAQPage JSON-LD を react-helmet-async で注入
3. 各大会ページの Event → SportsEvent スキーマ化
4. 中文ページ（`/zh/faq` 等）に中文FAQをミラー
5. sitemap にFAQページを追加
